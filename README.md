# LSPD METRO Division Website

## Lokal starten

```bash
npm install
npm run dev
```

Dann die angezeigte localhost-URL öffnen.

## Build testen

```bash
npm run build
```

## Auf Vercel hosten

1. Projekt auf GitHub hochladen.
2. Auf https://vercel.com einloggen.
3. New Project auswählen.
4. GitHub-Repository importieren.
5. Framework: Vite
6. Build Command: `npm run build`
7. Output Directory: `dist`
8. Deploy klicken.

## Wo Code bearbeiten?

Die Website liegt in:

`src/App.jsx`

Die Startdatei liegt in:

`src/main.jsx`

Das Basis-HTML liegt in:

`index.html`
